書影一括取込

書影情報を一括登録します。（登録済みの書影については上書きされます）
書影のファイルは [ISBN].jpg というファイル名でご用意いただき
アップロードする書影ファイルを全て選択した状態でzip形式に圧縮してください。
(zipを解凍した際にフォルダができるのではなく 書影ファイルがその場に展開される状態)

※書影ファイルは100点以内、zipファイルは50MB以内でご用意下さい。
